крч устанаваливай питон с офиц сайта, при установке ставь галочку "добавить в PATH"
открывай командную строку и выполни команды по порядку:
pip install -r libs.txt

в дискорде над создать сервер, а в нем вебхук (гугли). Ссылку вебхука надо вставить в файл pass.txt
чтоб запустить, кликай на hidden.vbs

Esc - закрывает скрипт
s - отправляет скрин
b - сообщение "не то"
l - "мало"
y - "да"
n - "нет"
